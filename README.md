#  Student Data Analysis Project

Analyze student performance using Python, Pandas, and NumPy.

* Python

* NumPy

* Calculated average scores
* Ranked students
* Identified topper
* Grouped data by ethnicity
* Analyzed performance trends

## 📈 Key Insights

* Identified top-performing students
* Found performance differences across groups

## Conclusion

This project demonstrates basic data analysis skills using Pandas and NumPy.
